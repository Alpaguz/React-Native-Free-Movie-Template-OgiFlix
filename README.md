# Project Title

Ogiflix - Free React Native Movie Template

## Description

A free to use template for your react native project.

## Getting Started

### Installing

* Clone repository
```
git clone https://github.com/Alpaguz/React-Native-Free-Movie-Template-OgiFlix.git
```
* Install node modules
```
npm install
```

### Executing program
```
npx expo start
```
